# dmenu - dynamic menu
dmenu is an efficient dynamic menu for X.
This is my fork of [suckless dmenu](https://tools.suckless.org/dmenu/).


# Requirements
In order to build dmenu you need the Xlib header files.


# Installation
```bash
sudo make clean install
```
